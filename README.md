# bumblebee-bot
Generate realtime driver bot with random trips.

## Requirements

[Node.js](https://nodejs.org/en/), tested with version 7.0.0.
We recomand the utilisation of [nvm](https://github.com/creationix/nvm).

## Instalation

    npm install

## Launch

    npm start
